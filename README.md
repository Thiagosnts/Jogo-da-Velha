# Jogo da velha Socket
Jogo da velha usando websocket, HTML e JS.

## Bibliotecas
- uuidv4
- socket.io

## Executar
- yarn
- node server/app.js
- acesse localhost:3000

## Rodando com docker
Criar imagem docker
`docker build -t jogo .`

Rodar imagem docker
`docker run -p 80:80 -d jogo`
