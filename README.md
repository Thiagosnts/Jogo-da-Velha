# Jogo da velha Socket
Jogo da velha usando websocket, HTML e JS.

## Bibliotecas
- uuidv4
- socket.io

## Executar
- yarn
- node server/app.js
- acesse localhost:3000

## rodando com docker
- crir imagem docker
docker build -t jogo .

- rodar imagem docker
docker run -p 80:80 -d jogo
